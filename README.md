# Task_Manager_SaaS_FullMaster
Командный task-менеджер с Kanban-доской, проектами и аналитикой. Стильный и кайфовый. Startup MVP edition.
Сайт с богатым UI, адаптивом, зазыванием пользователей и возможностями для работы с проектами/карточками. Плюсом идут тесты.

## Структура страниц

### Основные страницы
**index.vue** - Главная страница с выбором проекта, списком задач и Kanban-доской для управления задачами с drag-and-drop.

**board.vue** - Отдельная страница Kanban-доски (legacy).

**profile.vue** - Базовая страница профиля пользователя.

### Лендинг (landing/)
**main.vue** - Главный лендинг с hero-секцией, карточками преимуществ и зазыванием на регистрацию.

**about.vue** - Информационная страница о нас, продукте и его возможностях.

### Авторизация (auth/)
**login.vue** - Страница входа в систему с полями email и пароля, стильным дизайном и валидацией.

**register.vue** - Регистрация нового пользователя с проверкой имени (буквы, пробелы, апострофы, дефисы) и пароля (минимум 8 символов).

**test_create.vue** - Техническая страница для создания тестовых пользователей в dev-окружении.

### Профиль (profile/)
**index.vue** - Список приглашений в проекты с возможностью принять или отклонить запросы на вступление.

**my_projects.vue** - Управление своими проектами: создание, редактирование, удаление, приглашение участников и просмотр членов команды.

**analytics.vue** - Графики и статистика по задачам: распределение по статусам, количество горящих и просроченных задач.

### Layouts
**default.vue** - Основной layout с header'ом, навигацией, информацией о пользователе и счетчиком горящих задач.

**auth.vue** - layout для страниц авторизации без навигации (для будущих расширений).
